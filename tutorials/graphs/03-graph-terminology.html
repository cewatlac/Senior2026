Adjacency

Two nodes u and v are adjacent (or neighbors) if there is an edge between them. An edge is incident to both endpoints it connects.

Undirected: Neighbors of a node

In an undirected graph, if edge (u, v) exists then v is a neighbor of u and u is a neighbor of v. Click any node to highlight all its neighbors.

Directed: Out-neighbors vs In-neighbors

In a directed graph: out-neighbors of u = nodes v where u→v exists. In-neighbors of u = nodes w where w→u exists.

Degree (Undirected Graph)

The degree of a node is the number of edges incident to it — equivalently, the number of its neighbors. A node with degree 0 is called an isolated node. A node with degree 1 is called a leaf.

Handshaking Lemma: sum of all degrees = 2 × |E|
Consequence: number of odd-degree nodes is always even

In-degree & Out-degree (Directed Graph)

In a directed graph each node has two separate counts: in-degree = number of arrows pointing into the node, out-degree = number of arrows going out of the node.

Σ in-degrees = Σ out-degrees = |E|
Source node: in-degree = 0
Sink node: out-degree = 0

Handshaking Lemma

For any undirected graph: the sum of all node degrees equals twice the number of edges. Every edge contributes +1 to the degree of both its endpoints, so it's counted twice in the total.

Σ deg(v) = 2 × |E|
Useful to verify your graph: if sum of degrees is odd → you have a bug
Number of nodes with odd degree is always even
Verify: sum of all degrees = 2 × edges — hover to check each node's degree

Path vs Walk vs Cycle

A Walk is any sequence of nodes connected by edges — nodes and edges may repeat.

A Path is a walk with no repeated nodes (and therefore no repeated edges).

A Cycle is any closed walk that starts and ends at the same node — intermediate nodes may repeat.

A Simple Cycle is a closed walk where no intermediate node repeats — only the start and end are the same node. This is the common meaning in most graph problems.

Path: 0 → 1 → 2 → 3 (no repeats, open)
Walk: 0 → 1 → 5 → 0 → 1 (node 0 and 1 repeat)
Cycle: 0 → 1 → 0 → 2 → 0 (closed, nodes repeat — still a cycle)
Simple Cycle: 0 → 1 → 2 → 3 → 0 (closed, no intermediate repeats)

Subgraph & Spanning Subgraph

A subgraph of G is a graph H whose nodes and edges are all subsets of G's nodes and edges. A spanning subgraph includes all nodes of G but only some of its edges. A spanning tree is a spanning subgraph that is also a tree (connected, acyclic, n−1 edges).

Spanning tree of a graph with n nodes has exactly n−1 edges
Every connected graph has at least one spanning tree
MST (Minimum Spanning Tree) = spanning tree with minimum total edge weight → Kruskal or Prim
